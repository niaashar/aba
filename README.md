## 🚀 Установка

### 1. Загрузка на хостинг
1. Скопируйте все файлы в корневую папку сайта
2. Убедитесь, что структура папок сохранена
3. Проверьте права доступа (644 для файлов, 755 для папок)

### 2. Настройка изображений
1. Замените `images/Лого.png` на ваш логотип
2. Замените `images/Портфолио_ФИО.jpg` на фото специалиста
3. Замените `images/globus (1).png` на изображение глобуса

### 3. Документы
1. Создайте PDF-файлы из текстов выше
2. Загрузите в папку `documents/`
3. Проверьте ссылки в `contacts.html`
